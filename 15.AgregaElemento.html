Objetivo de la página web

Demostrar la manipulación dinámica de listas HTML mediante JavaScript, permitiendo agregar nuevos elementos de forma interactiva.

Funcionalidad

  • Muestra una lista inicial con 3 elementos
  • Permite agregar nuevos elementos a la lista con el botón "Añadir elemento"
  • Cada nuevo elemento se numera secuencialmente
  • La actualización de la lista se realiza sin recargar la página

Funciones JavaScript

  • añade() Función que:
    • Incrementa el contador de elementos
    • Crea un nuevo elemento de lista
    • Añade el nuevo elemento al final de la lista existente
  • document.createElement() Crea nuevos elementos HTML
  • innerHTML Propiedad que modifica el contenido HTML
