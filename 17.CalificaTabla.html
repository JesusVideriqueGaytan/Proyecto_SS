Operación Resultado Calificacion


Objetivo de la página web

Generar dinámicamente tablas de multiplicar seleccionables por el usuario, con funcionalidad de evaluación y calificación de resultados.

Funcionalidad

  • Permite seleccionar una tabla de multiplicar (del 2 al 10) mediante un menú desplegable
  • Al hacer clic en "Ejecutar", muestra la tabla seleccionada en formato de operaciones matemáticas
  • Incluye un sistema de calificación que evalúa los resultados mostrados
  • La tabla generada contiene tres columnas: Operación, Resultado y Calificación
  • La sección de resultados permanece oculta hasta que se genera una tabla

Componentes principales

  • select (opTabla) - Menú desplegable para seleccionar la tabla (2-10)
  • Botón "Ejecutar" - Genera y muestra la tabla seleccionada
  • Tabla dinámica - Muestra las operaciones de multiplicación
  • Botón "Calificar Examen" - Evalúa y califica los resultados
  • div (contenido) - Contenedor que se hace visible al generar resultados

Funciones JavaScript

  • muestra(sValor, oTbl, oElemento) Función principal que:
    • Recibe tres parámetros: el número base (como cadena), la tabla HTML donde se insertarán los datos y el contenedor a mostrar.
    • Convierte el valor seleccionado a número y genera dinámicamente 10 filas con operaciones de multiplicar.
    • Cada fila incluye:
      • La operación (ej. "3 * 1").
      • Un campo de texto para ingresar el resultado.
      • Una celda para mostrar la calificación (se actualiza en la función califica).
    • Si ya existía contenido en la tabla, primero elimina las filas anteriores.
    • Hace visible la sección de resultados.
  • califica(sValor, oTbl) Función que:
    • Recibe dos parámetros: el número base y la tabla a evaluar.
    • Recorre los campos de respuesta y verifica si el valor ingresado es correcto.
    • Si la respuesta es correcta, cambia el estilo del campo de texto (color de borde y fondo verde).
    • Cuenta el número de respuestas correctas y muestra una alerta con la calificación total.
    • Si detecta errores de captura (campos no numéricos), muestra un mensaje de error.
  • Métodos DOM utilizados:
    • createElement() Crea nuevos elementos HTML dinámicamente.
    • appendChild() Añade los nuevos elementos creados a la tabla.
    • insertRow() / insertCell() Inserta nuevas filas y celdas en la tabla HTML.
    • getElementById() Accede a los campos de texto para evaluar respuestas.
    • deleteRow() Elimina filas de la tabla antes de volver a generar una nueva.
    • style Modifica el estilo CSS de los campos de texto para indicar respuestas correctas.
