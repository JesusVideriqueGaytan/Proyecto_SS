Programa que verifica si un número es par

Objetivo de la página web

El objetivo de esta página es demostrar el uso avanzado de funciones en JavaScript para verificar si un número es par o impar mediante interacción con el usuario, mostrando los resultados en la misma sección.

Funcionalidad

  • El usuario introduce un número en el campo de texto.
  • Al hacer clic en "Verificar", se muestra el resultado en el área de resultados dentro de la misma caja.
  • El botón "Limpiar resultados" elimina todos los resultados mostrados.
  • El campo de entrada se limpia automáticamente después de cada verificación.

Funciones JavaScript

  • esPAR(numero) Muestra el resultado en el área designada dentro de la caja.
  • parImpar(numero) Determina si el número es par o impar usando el operador módulo.
  • limpiar() Limpia el área de resultados.
