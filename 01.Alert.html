Objetivo de la página web

El objetivo de esta página es mostrar el uso de archivos externos (como archivos CSS y JavaScript) en conjunto con una estructura básica de HTML. También busca demostrar la ejecución de mensajes emergentes (alerts) usando JavaScript.

Funcionalidad

1. Al cargar la página, se ejecuta un archivo externo de JavaScript (`codigo.js`) que muestra dos mensajes emergentes con la función `alert()`.

  • primer mensaje muestra: "Hola mundo".
  • Después, aparece otro mensaje que dice: "Soy el primer script".

2. Además, incluye estilos externos desde un archivo CSS (`estiloblog.css`) que aplica formato al contenido de la página. Aunque no se especifica el diseño, sabemos que el archivo debe modificar la apariencia visual.

3. También se incluye un mensaje en HTML que informa que la página requiere que JavaScript esté activado en el navegador.

Funcion JavaScript

  • alert() Muestra un cuadro de diálogo emergente en el navegador con el texto especificado dentro de los paréntesis.
