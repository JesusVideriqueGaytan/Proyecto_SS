Programa que verifica si un número es par

Objetivo de la página web

El objetivo de esta página es demostrar el uso de funciones en JavaScript para verificar si un número es par o impar mediante interacción con el usuario.

Funcionalidad

  • El usuario introduce un número en el campo de texto.
  • Al hacer clic en el botón "Verificar", se ejecuta la función esPAR().
  • La función llama a parImpar() que realiza el cálculo matemático.
  • Muestra un alert con el resultado (par o impar).

Funciones JavaScript

  • esPAR - Recibe el número, llama a parImpar() y muestra el resultado.
  • parImpar - Realiza la operación módulo (%) para determinar si es par o impar.
