Análisis de Cadenas de Texto

Esta página muestra análisis de tres cadenas de ejemplo mediante alertas.

Objetivo de la página web

El objetivo de esta página es analizar cadenas de texto para determinar si están formadas solo por mayúsculas, solo por minúsculas o por una combinación de ambas.

Funcionalidad

  • Al cargar la página, se ejecuta automáticamente un script que analiza tres cadenas predefinidas.
  • Cada análisis se muestra en una ventana emergente (alert) separada.
  • El script utiliza la función info() para realizar el análisis.

Funciones JavaScript

  • info(cadena) - Recibe una cadena y devuelve un mensaje indicando:
    • Si la cadena contiene solo mayúsculas
    • Si contiene solo minúsculas
    • Si es una mezcla de ambas
  • alert() - Muestra los resultados en ventanas emergentes
  • toUpperCase() - Convierte la cadena a mayúsculas
  • toLowerCase() - Convierte la cadena a minúsculas
