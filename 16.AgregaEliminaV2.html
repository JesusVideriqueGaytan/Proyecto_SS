Objetivo de la página web

Demostrar diferentes métodos de manipulación del DOM para agregar y eliminar elementos de una lista, comparando el enfoque DOM puro con la modificación directa del HTML.

Funcionalidad

  • Muestra una lista inicial con 5 elementos
  • Permite agregar elementos mediante dos métodos diferentes:
    • Método DOM: Usando createElement y appendChild
    • Método HTML: Modificando directamente innerHTML
  • Permite eliminar el último elemento de la lista
  • Todas las operaciones se realizan sin recargar la página

Funciones JavaScript

  • modifica(tipo) - Función principal que:
    • Recibe un parámetro que determina la operación (D, H o R)
    • Usa una estructura switch para ejecutar diferentes operaciones
  • Métodos DOM utilizados:
    • createElement() Crea nuevos elementos HTML
    • createTextNode() Crea nodos de texto
    • appendChild() Añade elementos al DOM
    • getElementsByTagName() Obtiene elementos por etiqueta
    • removeChild() Elimina elementos del DOM
    • innerHTML Modifica contenido HTML directamente
