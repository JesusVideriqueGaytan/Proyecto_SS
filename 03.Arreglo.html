Esta página muestra los 12 meses del año

Objetivo de la página web

El objetivo de esta página es demostrar el uso de arrays en JavaScript para almacenar y mostrar información, específicamente los nombres de los 12 meses del año.

Funcionalidad

1. Al cargar la página, se ejecuta un script embebido que:

  • Crea un array llamado "meses" con los 12 meses del año.
  • Muestra cada mes mediante 12 alertas consecutivas.

2. Incluye estilos externos desde un archivo CSS (`estiloblog.css`).

3. Muestra un mensaje en HTML que informa que la página requiere que JavaScript esté activado en el navegador.

Funciones JavaScript

  • Arrays Estructura de datos que almacena múltiples valores en una sola variable.
  • alert() Muestra cuadros de diálogo emergentes con cada uno de los meses.
