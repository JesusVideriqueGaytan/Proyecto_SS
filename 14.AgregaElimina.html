Objetivo de la página web

Demostrar el manejo dinámico de tablas HTML mediante JavaScript, permitiendo agregar y eliminar filas de forma interactiva.

Funcionalidad

  • Permite agregar nuevas filas a la tabla con el botón "Agregar"
  • Cada fila nueva contiene celdas con texto y un campo de entrada
  • El botón "Eliminar" remueve la última fila agregada
  • La manipulación de la tabla se realiza sin recargar la página

Funciones JavaScript

  • agregaLinea() Añade una nueva fila a la tabla con:
    • Una celda con texto fijo ("Celda")
    • Una celda con texto variable y un campo de entrada
    • Una celda con texto de confirmación ("agregado")
  • eliminaLinea() Elimina la última fila de la tabla
  • insertRow() Método para añadir filas a la tabla
  • insertCell() Método para añadir celdas a una fila
  • deleteRow() Método para eliminar filas de la tabla
