Esta página muestra un mensaje complejo

Objetivo de la página web

El objetivo de esta página es mostrar el uso de scripts embebidos en HTML junto con una estructura básica de la página. También busca demostrar la ejecución de mensajes emergentes (alerts) complejos usando JavaScript, incluyendo saltos de línea y diferentes tipos de comillas.

Funcionalidad

1. Al cargar la página, se ejecuta un script embebido en el head que muestra un mensaje emergente complejo con la función `alert()`.

  • El mensaje muestra: "Hola Mundo!" seguido de un salto de línea.
  • Continúa con: "Qué facil es incluir 'comillas simples'" y otro salto de línea.
  • Finaliza con: "y "comillas dobles""

2. Además, incluye estilos externos desde un archivo CSS (`estiloblog.css`) que aplica formato al contenido de la página.

3. También se incluye un mensaje en HTML que informa que la página requiere que JavaScript esté activado en el navegador.

Funcion JavaScript

  • alert() Muestra un cuadro de diálogo emergente en el navegador con el texto especificado, permitiendo incluir saltos de línea (\n) y diferentes tipos de comillas escapadas.
