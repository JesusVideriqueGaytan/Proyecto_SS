Cálculo de Factorial

Objetivo de la página web

El objetivo de esta página es calcular el factorial de un número introducido por el usuario.

Funcionalidad

  • Al hacer clic en el botón, se solicita un número mediante cuadro de diálogo.
  • Se calcula el factorial del número introducido.
  • El resultado se muestra en la página con el proceso detallado.
  • Incluye validación para números negativos y valores no numéricos.

Funciones JavaScript

  • for Multiplica los valores desde 1 hasta el número ingresado.
  • prompt() - Función que muestra un cuadro de diálogo para entrada de datos
  • alert() - Función que muestra el resultado en un cuadro de diálogo
